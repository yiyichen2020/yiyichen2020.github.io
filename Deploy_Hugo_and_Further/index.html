Deploy Hugo and Further

The following operations is based on the post, “Hugo With LoveIt Theme Configuration”, which requires you have already installed and config both Hugo and LoveIt theme.

1. Deploy Hugo on Github

  1. Assume you have already got a Github account.

  2. Create a Github Repository, name this repository as yourgithubname.github.io.

  3. Open your terminal, and cd to your blog folder.

  4. Run following command lines:

1
2
3
4
5
6
7
8

hugo --theme=loveit --baseUrl="https://yourgithubname.github.io/"
 # To build static pages, which will generate public folder
cd public
git init
git remote add origin https://github.com/yourgithubname/yourgithubname.github.io.git
git add .
git commit -m "first commit"
git push
  1. After a few seconds, your blog is online!

2.Advance

1. Add Comment Box to Your Blog

  1. Install utterances on your repo.

The easiest way for you to add comment box is adding utterances.es comment box. Here is the brief intruduction of utterances.es: https://utteranc.es/

  1. Open quickstart/config.toml file, and set utterances.es related setting like following.
1
2
3
4
5
6
7
8

[params.page.comment.utterances]
    enable = true
    # owner/repo
    repo = "yourgithubname/yourgithubname.github.io"
    issueTerm = "pathname"
    label = ""
    lightTheme = "github-light"
    darkTheme = "github-dark"
  1. Set enable = false under [params.page.comment.valine]
  2. Rebuild your blog and redeploy your public folder.
  3. Enjoy commenting🍻

2. Add Advanced Search (Algolia) Ability to Your Blog

  1. Create algolia account: https://www.algolia.com/

  1. Create an application under your algolia account; [Choose Experienced User, continue all the way down]

  1. Configure your application; [Click Indice in the left sidebar, click Create Index in the right sidebar just pop out, choose a name you like as Index name]

  1. Add your algolia application related API Keys into your config.toml file. You can find API Key on the sidebar in algolia website.

[languages.en.params.search]
        enable = true
        # 搜索引擎的类型 ("lunr", "algolia")
        type = "algolia"
        # 文章内容最长索引长度
        contentLength = 4000
        # 搜索框的占位提示语
        placeholder = ""
        # 最大结果数目
        maxResultLength = 10
        # 结果内容片段长度
        snippetLength = 50
        # 搜索结果中高亮部分的 HTML 标签
        highlightTag = "em"
        # 是否在搜索索引中使用基于 baseURL 的绝对路径
        absoluteURL = true
        [languages.en.params.search.algolia]
          index = "application-name"
          appID = "Application ID"
          searchKey = "Search-Only API Key"
      # 主页信息设置
  1. Uncomment home = ["HTML", "RSS", "JSON"] under [outputs] in your config.toml file

  1. Cd to your blog directory and Rebuild Hugo Blog. You can find there is a index.json file under quickstart/public folder. Upload index.json to your algolia application

  1. Redeploy your site, enjoy searching!
