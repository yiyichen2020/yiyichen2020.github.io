Deploy Hugo and Further

The following operations is based on the post, “Hugo With LoveIt Theme Configuration”, which requires you have already installed and config both Hugo and LoveIt theme.

1. Deploy Hugo on Github

  1. Assume you have already got a Github account.

  2. Create a Github Repository, name this repository as yourgithubname.github.io.

  3. Open your terminal, and cd to your blog folder.

  4. Run following command lines:

1
2
3
4
5
6
7
8

hugo --theme=loveit --baseUrl="https://yourgithubname.github.io/"
 # To build static pages, which will generate public folder
cd public
git init
git remote add origin https://github.com/yourgithubname/yourgithubname.github.io.git
git add .
git commit -m "first commit"
git push
  1. After a few seconds, your blog is online!

2.Advance

1. Add Comment Box to Your Blog

  1. Install utterances on your repo.

The easiest way for you to add comment box is adding utterances.es comment box. Here is the brief intruduction of utterances.es: https://utteranc.es/

  1. Open quickstart/config.toml file, and set utterances.es related setting like following.
1
2
3
4
5
6
7
8

[params.page.comment.utterances]
    enable = true
    # owner/repo
    repo = "yourgithubname/yourgithubname.github.io"
    issueTerm = "pathname"
    label = ""
    lightTheme = "github-light"
    darkTheme = "github-dark"
  1. Set enable = false under [params.page.comment.valine]
  2. Rebuild your blog and redeploy your public folder.
  3. Enjoy commenting🍻
