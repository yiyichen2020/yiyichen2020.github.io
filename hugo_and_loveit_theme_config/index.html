Hugo With LoveIt Theme Configuration

This post is for MacOS and Homebrew User

0. Install Homebrew

1

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

1. Install Hugo

# Install hugo by homebrew
brew install hugo
# Make sure that you install hugo successfully
hugo version
# Cd to Desktop directory
cd Desktop
# Create new hugo site in Desktop directory
hugo new site quickstart

Install LoveIt Theme

1

9

# Cd to quickstart directory
cd quickstart
# init your own git repository
git init
# Install LoveIt theme from github
git submodule add https://github.com/dillonzq/LoveIt.git themes/LoveIt
# add and commit your site
git add .
git commit -m "Added LoveIt Theme"

3. Usages of Hugo

3-0 Add Your Avatar Image

If you are following every step above this post, so your Avatar Image should be underquickstart/static/images folder. Also, naming your avatar image as avatar.jpg

3-1 Create Your First Post

1

hugo new posts/first_post.md

The default content in first_post.md:

1
2
3
4
5

---
title: "{{ replace .Name "-" " " | title }}"
date: {{ .Date }}
draft: true
---

The default content is generated by quickstart/archetypes/default.md, you can change the default content in that file!

By default all posts and pages are created as a draft. So this first_post will not be shown in your site. Please do change draft: true to draft: false

    {{- $content = partial "function/ruby.html" $content -}}
{{- end -}}

    {{- $content = partial "function/fraction.html" $content -}}
